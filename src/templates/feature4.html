AWS CloudWatch 告警設定指南

EC2 整合教學

當您將本專案部署到 EC2 後，請依照下方步驟設定 AWS 雲端警告，本頁面的燒機按鈕將會立刻觸發該警報：

1

建立 CloudWatch 告警指標 (Alarm)

前往 AWS 主控台 -> 搜尋 CloudWatch -> 點選「告警 (Alarms)」 -> 建立告警指標，並選擇 EC2 > Per-Instance Metrics -> 指標搜尋當前 EC2 實例的 CPUUtilization。

2

設定負載條件與監控週期

閾值條件設定為「大於/等於 (>=)」 -> 輸入 80%，並將統計間隔設為「1 分鐘」或「5 分鐘」（配合您左側選擇的燒機持續時間），確保告警能靈敏捕捉負載爆衝。

3

配置通知與觸發 Amazon SNS

在動作設定 (Actions) 選擇「警報狀態 (In Alarm)」，觸發目標選擇新建立或現有的 Amazon SNS 主題，設定您的電子郵件或 Slack webhook 接收 CloudWatch 自動發送的警告通知郵件！
